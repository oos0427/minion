CSS 시작 

CSS란?

CSS(Cascading Style Sheets)는 웹 페이지의 시각적 스타일을 정의하는 언어입니다. HTML로 웹 페이지의 구조를 만들고, CSS로 그 구조에 스타일을 적용합니다. 예를 들어, 글꼴의 색상, 크기, 배경색, 레이아웃 등을 조정할 수 있습니다.
CSS는 웹 디자인을 더 아름답고 기능적으로 만들 수 있는 중요한 도구입니다.

CSS는 주로 다음과 같은 기능을 제공합니다:

- 스타일 정의: 글꼴, 색상, 여백, 패딩, 테두리 등을 설정합니다.
- 레이아웃 조정: 요소의 배치와 크기를 조정합니다.
- 반응형 디자인: 다양한 화면 크기에 맞춰 스타일을 조정합니다.
- 애니메이션: 요소에 애니메이션 효과를 추가할 수 있습니다.

스타일 
- 태그선택자 h2 와 같은 태그를 가진 객체는 모두 같은 스타일을 가지게함
- 고유아이디선택자 고유아이디를 할당하여 그 아이디를 부여한 객체에 같은 스타일을 가지게함

스타일 파일은 따로 만들어서 관리할 수도 있으며 여러 사람이 협업할 때 이미지 스타일 html 따로 관리하면 편함
