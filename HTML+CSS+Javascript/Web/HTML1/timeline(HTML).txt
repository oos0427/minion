<input type="range" min="0" max="1000" step="100" value="100"/>
다음은 각 속성에 대한 설명입니다:
min="0": 슬라이더의 최소값을 0으로 설정
max="1000": 슬라이더의 최대값을 1000으로 설정
step="10": 슬라이더의 값을 10 단위로 조정
value="10": 슬라이더의 초기값을 10으로 설정
